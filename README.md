## About This Repository

Welcome to the **AI Solutions Developer** course learner repository. This is your central hub for all course materials, updated and maintained by your instructor.

Each module folder contains **Markdown (.md)** files you can read directly on GitHub with fully clickable links. These files include:

- **Lesson files** with key concepts, learning objectives, and reference material reconstructed from course slides
- **Pacesetter Guides** with instructions, learning objectives, and direct links to your DataCamp courses
- **POC (Proof of Completion)** submission instructions with badge requirements
- **GLAB (Guided Lab)** step-by-step instructions
- **All Links** quick reference files with every link you need for each module

---

## Important Information for Learners

### DataCamp Access

Most of your coursework is completed on the **[DataCamp](https://app.datacamp.com/)** learning platform. Direct links to each DataCamp course are provided in the pacesetter guide files and all_links files in each module folder. If you did not receive your DataCamp invitation, contact your instructor or TA immediately.

### Canvas Links

Canvas assignment submission links are included in the files for reference. You will need to be logged into Canvas to access them. **Use the links in this repo** as your primary reference since they are verified and organized by module.

### Supplementary Resources

Each pacesetter guide includes supplementary resources (articles and videos) to deepen your understanding of the topic. These are not graded but are highly recommended.

## How to Use

1. **Browse on GitHub** - Navigate to any module folder and click on `.md` files to read them with clickable links.
2. **Start with all_links.md** - Each module has an `all_links.md` file with every link you need in one place.
3. **Follow Pacesetter Guides** - These guide you through each DataCamp course section with context and learning objectives.
4. **Submit POCs** - Follow the POC files for badge submission instructions.
5. **Download Everything** - Click the green **Code** button above, then **Download ZIP**, or clone:
   ```bash
   git clone https://github.com/akarales/UCI_3097_AI_Solutions_Developer_public.git
   ```
