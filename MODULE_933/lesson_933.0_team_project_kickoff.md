Version 1.0, May 2026

## Overview

This lesson kicks off the AI Solutions Developer course with team formation, introductions, and an overview of the 13-week program. You will learn about the course structure, expectations, technology stack, and career outcomes.

## Learning Objectives

- Understand the course structure, modules, and grading criteria
- Meet your team members and establish collaborative workflows
- Learn about the technology stack used throughout the course
- Understand assessment types: KBA, SBA, Capstone, and ICP

---

## Course Structure

### 13-Week Program (117 Sync Hours)

| Module | Topic | Hours | Weight |

| **933** | Introduction to AI & Tools | 6 hrs | 10% |
| **937** | Foundations & Large Language Models | 15 hrs | 13% |
| **938** | Prompt Engineering & Orchestration | 24 hrs | 15% |
| **939** | Retrieval-Augmented Generation (RAG) | 21 hrs | 13% |
| **940** | Agentic AI Solutions | 12 hrs | 12% |
| **941** | Multimodality | 12 hrs | 12% |

In-Class Participation (ICP) accounts for 5% of your final grade.

## Technology Stack

### Programming & Development
- Python 3.10+
- API Integration and Request Handling
- Functions and Object-Oriented Design

### AI Platforms & Models
- OpenAI APIs and SDKs
- Hugging Face Hub (models, pipelines, inference)
- Ollama and GPT4All (local LLM runtimes)
- Prompt Engineering Techniques

### Frameworks & Orchestration
- LangChain (with Mistral and LLaMA models)
- Tool and Function Calling
- Vector Databases (Pinecone, ChromaDB)
- Embedding Models and Retrieval Pipelines

### Learning Platforms
- **Canvas LMS** - Assignments, assessments, schedules, and course materials
- **[DataCamp](https://app.datacamp.com/)** - Guided self-study and hands-on labs

## Assessment Types

| Type | Format | Description |
|------|--------|-------------|
| **KBA** | Closed-book, MCQ, on-camera | Knowledge-based assessments testing concept understanding |
| **SBA** | Open-book, programming/projects | Skill-based assessments testing practical ability |
| **Capstone** | Comprehensive project (20%) | Portfolio-ready AI application demonstrating all skills |
| **ICP** | Ongoing (5%) | Communication, comprehension, teamwork, punctuality |

## Career Outcomes

Upon successful completion, you will be equipped for roles such as:

- **AI Solutions Developer** - Design, build, and deploy complete AI applications
- **Full-Stack Engineer (AI-Focused)** - Add AI features to web apps and intelligent UX
- **ML/AI Engineer** - Build production ML workflows with RAG and agents
- **AI Product Developer** - Create AI products with multimodal capabilities

---

## Keys to Success

- **Consistency** - Attend all sessions, complete assignments on schedule, build daily coding habits
- **Hands-On Practice** - Learn by building, experiment beyond assignments, debug your own errors
- **Collaboration** - Ask questions, participate in discussions, help peers
- **Portfolio Mindset** - Document your learning, treat each project as showcase-worthy
