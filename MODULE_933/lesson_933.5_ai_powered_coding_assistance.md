## Overview

This lesson explores the broader landscape of AI coding assistants beyond GitHub Copilot. You will learn about the different tools available, their strengths and weaknesses, and how modern developers and analysts integrate AI into their daily workflows.

## Learning Objectives

- Compare major AI coding assistants available today
- Understand how AI tools integrate into different development workflows
- Evaluate when to use AI assistance and when to code manually
- Apply AI-assisted development to data analysis tasks

---

## Section 1: The AI Coding Assistant Landscape

### Major AI Coding Assistants (2025-2026)

| Tool | Provider | Key Strength | Best For |
|------|----------|-------------|----------|
| **GitHub Copilot** | GitHub/OpenAI | Deep IDE integration | General development |
| **Cursor** | Cursor Inc. | AI-native IDE | Full project context |
| **Windsurf (Cascade)** | Codeium | Agentic coding | Multi-file edits |
| **Amazon CodeWhisperer** | AWS | AWS integration | Cloud development |
| **Cody** | Sourcegraph | Codebase search | Large codebases |
| **ChatGPT/Claude** | OpenAI/Anthropic | Conversational | Learning and exploration |
| **Google Gemini** | Google | Multimodal | Code + visual tasks |

### How These Tools Differ

- **Inline Completion** - Copilot, CodeWhisperer (suggest as you type)
- **Chat-Based** - ChatGPT, Claude (conversational coding assistance)
- **Agentic** - Cursor, Windsurf (can make multi-file changes autonomously)
- **Search-Based** - Cody (searches your entire codebase for context)

## Section 2: AI-Assisted Development Workflow

### When to Use AI Assistance

- **Boilerplate code** - Repetitive patterns, config files, setup code
- **Learning new APIs** - Get quick examples and explanations
- **Debugging** - Describe the error and get suggested fixes
- **Documentation** - Generate docstrings, README content, comments
- **Testing** - Create unit tests for existing functions
- **Refactoring** - Suggest cleaner implementations

### When to Code Manually

- **Core business logic** - Critical code that must be correct
- **Security-sensitive code** - Authentication, encryption, data handling
- **Performance-critical sections** - When optimization matters
- **Novel algorithms** - When the problem is unique to your domain

## Section 3: AI for Data Analysis

Data analysts and data scientists use AI tools differently than software engineers:

### Common AI-Assisted Analysis Tasks

1. **Data Cleaning** - "Write pandas code to handle missing values in this dataframe"
2. **Exploratory Analysis** - "Create a correlation matrix and heatmap for these columns"
3. **Visualization** - "Generate a matplotlib chart showing monthly trends"
4. **SQL Queries** - "Write a SQL query to find the top 10 customers by revenue"
5. **Statistical Tests** - "Perform a t-test comparing these two groups"
6. **Report Generation** - "Summarize these findings in a markdown report"

### Google Colab + AI

Google Colab provides a free, cloud-based Jupyter notebook environment that integrates well with AI assistants. You can:
- Write and run Python code in the browser
- Access free GPU/TPU resources
- Share notebooks with collaborators
- Use AI to generate code cells
