## Section 1: What is Generative AI?

Generative AI refers to AI models that can create new content, including text, images, audio, code, and more. Unlike traditional AI that classifies or predicts based on existing data, generative AI produces novel outputs.

### Key Concepts

- **Generative vs. Discriminative Models** - Generative models create new data, discriminative models classify existing data
- **Foundation Models** - Large pre-trained models that can be adapted to many tasks
- **Large Language Models (LLMs)** - Foundation models specifically designed for text generation
- **Prompts** - The input text you provide to guide the model's output
- **Tokens** - The units of text that LLMs process (roughly 4 characters per token)

### How LLMs Generate Text

1. Input text is tokenized (broken into tokens)
2. Tokens are converted to numerical embeddings
3. The model processes embeddings through transformer layers
4. Each layer applies self-attention and feed-forward operations
5. The model predicts the most likely next token
6. The process repeats until the response is complete

## Section 2: Applications of Generative AI

| Application | Description | Example |
|------------|-------------|---------|
| **Text Generation** | Creating written content | ChatGPT, Claude |
| **Code Generation** | Writing and debugging code | GitHub Copilot, Cursor |
| **Image Generation** | Creating images from text descriptions | DALL-E, Midjourney |
| **Summarization** | Condensing long documents | Meeting notes, article summaries |
| **Translation** | Converting between languages | Real-time translation services |
| **Q&A Systems** | Answering questions from documents | RAG-powered chatbots |
